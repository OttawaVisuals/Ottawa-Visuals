# Ottawa Visuals

**Live site:** [https://](https://ottawavisuals.github.io/Ottawa-Visuals/)

## Edit points
- Home page: `/index.md`
- Pages: `/pages/*.md`
- Images: `/assets/img/`
- Styles: `/assets/css/custom.css`
- Top nav: `/_config.yml` → `nav:`

## Power BI
Power BI → File → **Publish to web** → paste the iframe into your page.
